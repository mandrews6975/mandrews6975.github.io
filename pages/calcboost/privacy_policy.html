CalcBoost Privacy Policy

Data Collection By The Developer

The developer (Michael Andrews) does not directly collect, store or handle end user data in any way. CalcBoost is developed using Expo. Expo provides APIs that allow for simplified development of mobile apps. Some of these APIs contain code that allows for collection of end user data. These APIs are bundled with CalcBoost (like all Expo apps), but CalcBoost does not use these APIs to collect end user data.

Data Collection By Expo

As previously described, CalcBoost is developed using Expo. Expo provides a privacy policy that covers developers and end users of developers' apps. Expo also provides an explanation of their privacy policy to better explain the specifics of data they collect from developers and end users of developers' apps. Please open the links that are provided in the previous sentences to view Expo's privacy policy and privacy policy explanation.
